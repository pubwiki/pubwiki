# Your First Project

Let's create a simple AI interactive story step by step.

## Create a New Project

1. Open [Studio](https://studio.pub.wiki)
2. Click "New Project"
3. Give your project a name

## Add a Prompt Node

The Prompt node is the foundation of your AI interaction:

1. Right-click on the canvas
2. Select "Add Node" → "Prompt"
3. Write your system prompt, for example:

```
You are a friendly wizard in a magical forest. 
Guide the player through an adventure.
```

## Add an Input Node

Input nodes capture user responses:

1. Add an Input node
2. Connect it to your Prompt node
3. The input will be included in the AI context

## Add a Generated Node

Generated nodes display AI responses:

1. Add a Generated node
2. Connect it to receive the AI output
3. This will show the wizard's response

## Preview Your Story

1. Click the "Play" button
2. Interact with your AI wizard
3. See the magic happen!

## Save and Publish

When you're happy with your creation:

1. Click "Save" to save your progress
2. Click "Publish" to share with the community

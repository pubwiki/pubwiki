# Getting Started

This guide will help you get up and running with Pub.Wiki in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An internet connection

## Quick Start

1. Visit [studio.pub.wiki](https://studio.pub.wiki)
2. Create a new project
3. Start adding nodes to your canvas
4. Preview your creation in the sandbox

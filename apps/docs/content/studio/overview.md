# Studio Overview

Studio is the creative heart of Pub.Wiki - a visual node editor for building AI-powered interactive experiences.

## Interface

The Studio interface consists of:

- **Canvas** - The main workspace where you arrange nodes
- **Sidebar** - Tools and properties panel
- **Toolbar** - Quick actions and settings
- **Preview** - Live sandbox for testing

## Key Concepts

### Nodes

Nodes are the building blocks of your story. Each node type serves a specific purpose:

- **Prompt** - AI instructions and context
- **Input** - User interaction points
- **Generated** - AI-generated content
- **VFS** - Virtual file system for assets
- **Sandbox** - Live preview window
- **Loader** - Custom Lua/TypeScript scripts

### Connections

Nodes are connected with edges that define data flow. Connections are type-safe and validated.

### Version Control

Every change is tracked. You can:

- View history
- Create snapshots
- Revert to previous versions
- Fork and branch

# Studio

Studio is the creative heart of Pub.Wiki - a visual node editor for building AI-powered interactive experiences.

## What is Studio?

Studio provides a node-based visual editor where you can:

- Design AI conversation flows
- Create branching narratives
- Add custom logic with Lua/TypeScript
- Preview your creation in real-time

# Welcome to Pub.Wiki

Pub.Wiki is an open platform for creating, sharing, and playing AI-powered interactive stories.

## What is Pub.Wiki?

Pub.Wiki consists of three main components:

- **Studio** - A visual node editor for creating AI roleplay experiences
- **Hub** - A community platform for publishing and discovering artifacts
- **Sandbox** - A runtime environment for playing interactive stories

## Getting Started

Ready to start creating? Check out our [Getting Started](/getting-started) guide.

## Features

### Visual Node Editor

Design complex AI interactions without writing code. Our node-based editor supports:

- Prompt nodes for AI instructions
- Input nodes for user interaction
- Generated nodes for AI responses
- VFS nodes for managing game assets
- Sandbox nodes for live preview

### Cloud Save Sync

Your game progress is saved in the cloud. Continue your story on any device.

### Fork & Remix

Build on what others have created. Fork existing artifacts and make them your own.
